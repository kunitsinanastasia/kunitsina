Цель

Научиться создавать html страницы, нормированные списки и подключать к ним таблицу стилей.

Задачи

  1. Создать простейшую html страницу
  2. Создать нормированные списки
  3. Подключить таблицу стилей
  4. Написать отчет по лабораторной

План выполнения работы

    Сначала необходимо создать папку с проектом, в ней создать следующие файлы и папки:
  1. Файл "index.html". В этом файле будет располагаться основное содержимое страницы.
  2. Папку "css". Она предназначена для хранения в них таблиц стилей.
    1. Внутри папки "css" создать пустой файл "style.css".
  3. Папку "source". В дальнейшем в этой папке можно будет расположить скрипты на javascript

Ход выполнения работы

№ п/п Наименование Как реализовывалось
1 Заполнение "index.html" Было предложено совместить задачи 1. и 3. Таким образом отпадает необходимость придумывать актуальную тему, которой можно посвятить личную страницу. При выполнении работы используются простейшие теги html, такие как <head>, <body>, <h1>, <h2>, <h3> , <br>, <p>. Также были использованы специальные теги <meta> и <link>.
2 Заполнение "css/style.css" После первичного заполнения файла "index.html" можно подключить таблицу стилей через конструкцию <link rel="stylesheet" type="text/css" href="css/style.css>. По последним стандартам данную конструкцию следует вставлять перед </head>. Таблица стилей открывает широкие возможности по изменению внешнего вида страницы, однако в рамках данной работы я ограничился изменением размера текста, а также закрепил полотно.
